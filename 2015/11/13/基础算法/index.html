基础算法

By Hidden Feng

发表于 2015-11-13

在数学和计算机科学/算学之中，算法/算则法为一个计算的具体步骤，常用于计算、数据处理和自动推理。精准而言，算法是一个表示为有限长列表的有效方法。算法应包含清晰定义的指令用于计算函数。算法代表着用系统的方法描述解决问题的策略机制。

穷举法（枚举法）

原理： 根据一定的条件，列出问题的所有可能解，然后对所有情况进行验证，得到最后解或无解。

例： 设有A、B、C、D四个数都在1~16范围内。要求打印四个数都不相同时，其和为34的所有值。

算法分析：
最直接的想法就是设置A\B\C\D四个变量，都从1-16循环。需要161616*16次。且循环中还必须判断4个数是否有重复的情况。
但是题目提及四个数都不同，故，设A<B<C<D，A的初值为1，B的初值为A+1，C的初始值为B+1，D的初始值为C+1。
每次循环的最终以可以算出来：


A + (A+1) + (A+1+1) + (A+1+1+1) = 34     A = 7
1 + B + (B+1) + (B+1+1) = 34             B = 10
1 + 2 + C + (C+1) = 34                   C = 15
1 + 2 + 3 + D =34                        D = 28

算法局限：当一个问题的状态空间很大时，穷举法的计算量可能会太大。
